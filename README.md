# SpotTrader 1h Dashboard (Vercel)

## Vercel Deploy

1) Push this repo to GitHub.
2) Import it in Vercel.
3) Framework preset: Other.
4) Root directory: repo root.
5) Deploy.

This serves the static dashboard from `public/` and connects to the 1h backend at:
https://umarmuneeb451--spot-trader-bot-1h-run-dashboard.modal.run

## Vercel CLI (optional)
